# Personal Portfolio

A sleek, modern, and interactive personal portfolio website built to showcase my projects, skills, and experience. Designed with a focus on responsive layout, smooth animations, and a glassmorphism aesthetic.

## 🛠️ Built With

*   **[React 18](https://react.dev/)** - Frontend library
*   **[TypeScript](https://www.typescriptlang.org/)** - Static typing
*   **[Vite](https://vitejs.dev/)** - Frontend tooling and bundler
*   **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first styling framework
*   **[Framer Motion](https://www.framer.com/motion/)** - Production-ready animation library

## ✨ Features

*   **Modern UI/UX**: Sleek dark mode design with glassmorphism elements and glowing accents.
*   **Interactive Animations**: Scroll-triggered reveals, hover effects, and smooth transitions powered by Framer Motion.
*   **Responsive Design**: Fully functional across mobile, tablet, and desktop viewports.
*   **Project Showcase**: Dedicated section to highlight top projects with image carousels, skill tags, and direct links to live code and repositories.
*   **Dynamic Skills Grid**: Categorized technical skills presented with recognizable tech icons and staggered animations.
*   **Contact Form**: Integrated contact section ready for an email API service (like Web3Forms).

## 🏃‍♂️ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Hari17122005/your-repo-name.git
    ```
2.  Navigate to the project directory
    ```sh
    cd your-repo-name
    ```
3.  Install NPM packages
    ```sh
    npm install
    ```
4.  Run the development server
    ```sh
    npm run dev
    ```

## 📨 Contact Form Setup

The contact form is pre-configured to work with [Web3Forms](https://web3forms.com/). To enable it:

1.  Get your free access key from Web3Forms.
2.  Create a `.env` file in the root directory.
3.  Add your key:
    ```env
    VITE_WEB3FORMS_ACCESS_KEY=your_access_key_here
    ```
